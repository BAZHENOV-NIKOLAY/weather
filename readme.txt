В терминале пишем
1. cd "C:\weathher"- должен произойти переход в нушную папку
2. npm run dev - будет запущен наш сайт и появиться адрес сайта. Этот скрипт запускает локальный сервер
3. Далее переходим по адресу прописанном в терминале, чтобы открыть наш сайт
4. Если мы закрыли терминал то и сайт тоже закрывается. Ctrl+C -отключаемся от сервера
5. Далее вверху выбираем file->Open Folder
6. Выбираем наш проект и он начинает отображаться слева в дереве проектов. Запустить заново проект на сервере  npm run dev
